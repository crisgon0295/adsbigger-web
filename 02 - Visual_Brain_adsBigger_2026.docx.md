**06 — DIRECCIÓN DE IMAGEN**

| ESTÉTICA GENERAL Cinematográfica, oscura, alto contraste. Piensa en fotogramas de película, no en fotos de stock. Figuras aisladas, sombras duras, iluminación dramática. Las imágenes deben sentirse como una escena — quieta intensidad, no perfección posada. Gradación cálida con tonos desaturados. |
| :---- |

| Hook | Dramática y tensa. Figura única u objeto aislado. Alto contraste visual. Debe generar una segunda mirada antes de leer el texto. |
| :---: | :---- |
| **Valor** | Más calmada, con espacio. Puede usar texturas abstractas o detalles recortados. El foco está en el texto, la imagen es soporte. |
| **Tensión** | Regresa al drama. Movimiento, contraste, energía. El slide de tensión debe sentirse como el punto de inflexión de la historia. |
| **CTA** | Sin imagen. Solo color sólido de la paleta de marca (Obsidian o Rojo). Tipografía centrada. El CTA slide debe sentirse diseñado, no vacío. |

| USAR Fotografías con estética de film noir o cinematográfica. Iluminación lateral o frontal dura. Sombras pronunciadas. Gradación de color oscura y desaturada. Texturas industriales (concreto, metal, madera oscura). Imágenes de Cris: estilo natural, no posado, con contexto de trabajo. | NUNCA USAR Stock corporativo con personas sonriendo en traje. Colores neón brillantes. Ilustraciones caricaturescas. Fotos de alta exposición o filtros vintage cálidos extremos. Diseños con muchos elementos decorativos. Imágenes genéricas de 'éxito' (coches, relojes, jets). |

**09 — ARQUITECTURA DE CONTENIDO**

| S1 | HOOK Máx 8 palabras | Afirmación bold o pregunta que genera tensión. Space Grotesk 700 CAPS. Texto centrado. Imagen cinematográfica de fondo con overlay 55–65%. Máximo 8 palabras. Ejemplo: 'TU NEGOCIO NO NECESITA MÁS ADS.' |
| :---: | :---- | :---- |

| S2 | CONTEXTO Máx 15 palabras | Establece el problema real. Por qué existe el pain point. Sin solución todavía. Fondo claro, texto izquierda, cuerpo en Inter. Crea identificación antes de ofrecer la salida. |
| :---: | :---- | :---- |

| S3 | VALOR 1 Máx 15 palabras | Primera pieza de sustancia. Una sola idea por slide. Dato, principio o reframe. Fondo oscuro \+ imagen de soporte. Space Grotesk headline \+ Inter cuerpo. |
| :---: | :---- | :---- |

| S4 | VALOR 2 Máx 15 palabras | Segunda idea. Más profundidad. Fondo claro, solo texto. El lector ya confía — ahora recibe el cómo. |
| :---: | :---- | :---- |

| S5 | VALOR 3 Máx 15 palabras | Tercera idea o caso concreto. Puede incluir dato real, resultado del cliente, o componente del sistema. Fondo oscuro \+ imagen. |
| :---: | :---- | :---- |

| S6 | TENSIÓN Máx 12 palabras | El pivote. El slide más incómodo. Desafía una creencia. Usa el rojo. Alta energía visual. Es el punto de inflexión que justifica el CTA. Ejemplo: 'SI SIGUES PAGANDO AGENCIAS SIN SISTEMA, ESTÁS FINANCIANDO EL PROBLEMA.' |
| :---: | :---- | :---- |

| S7 | CTA Máx 6 palabras | Fondo sólido Obsidian. Space Grotesk 700 CAPS. Acción directa y concreta. Handle @adsbigger prominente en Azul o Rojo. Sin imagen. El CTA debe sentirse diseñado, nunca vacío. Tagline de cierre. |
