**REGLAS DE VOZ**

| ✓ | Habla siempre de sistemas, no de anuncios. La palabra SISTEMA aparece en cada titular si es posible. |
| :---: | :---- |
| **✓** | Usa el lenguaje exacto del cliente: predecible, estructura, activo, proceso, calificado, cierre, pipeline. |
| **✓** | Genera incomodidad productiva. Di la verdad incómoda primero, luego da la salida. |
| **✗** | Nunca uses métricas de vanidad. No: likes, alcance, impresiones. Sí: leads calificados, clientes cerrados, ROAS. |
| **✗** | Nunca suenes a agencia. Prohibido: 'estrategia integral', 'posicionamiento de marca', 'ecosistema digital'. |
| **✗** | Nunca promesas sin garantía. 'Clientes calificados cada semana' siempre va con '— o trabajamos gratis hasta lograrlo.' |
